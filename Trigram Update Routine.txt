Trigram Update Routine:

1) Select next Trigram (e.g. "TOU") *instructions below if new trigram is needed

2) Make json dict by running "python3 make_trigram_dict_json.py TOU" in the folder Desktop/Trigram\ Dict/code

3) Copy json dict ("tou_words.json") into Desktop/Trigram/data folder

4) Update calendar.js in Desktop/Trigram/js folder so that TOU is next in the trigram list

5) Push changes to main ("git add ." -> "git commit -m [message in quotes]" -> git push origin main)

6) Mark TOU as DONE in the Spreadsheet (link below) 

7) Make new slide in Powerpoint for "TOU" (link below). Update the trigram number, trigram, and date

8) Take a square-shaped screenshot of the TOU image in the slide, and save it to Desktop/Trigram\ Social as "trigram_announce_[trigram number].png"

9) On Sunday at ~midnight, make instagram post with this png, and the caption "Trigram #[trigram number] is now available! Play here: https://trigram.netlify.app

*If new trigram needs to be found:
1) Pick an un-triaged trigram (e.g. "ORD") from spreadsheet

2) Run "python3 getWords.py ORD" in the folder Desktop/Trigram\ Dict/code

3) Inspect the available words of length 4 through 15 to ensure it would make for a feasible/enjoyable game

4) If so, mark ORD as YES in the spreadsheet, else mark as NO
